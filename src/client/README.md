CStlProcessing.py
---------------------

This file allows the user to load and graph a .stl.  

Install the following libraries:
1. numpy
2. matplotlib
3. mpl_toolkits
4. stl

To install stl:  

    pip install numpy-stl

To process any __stl__, specify the .stl path in the variable *path*.
